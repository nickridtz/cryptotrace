Версия 2.2 — Авторизация и темизация
НовоеАвторизация — вход по логину и паролю, доступ к приложению только для зарегистрированных пользователей
НовоеПанель администратора — управление пользователями: добавление, блокировка, смена пароля
НовоеРоли — разграничение прав: администратор видит панель управления, обычный пользователь — нет
НовоеСветлая тема — переключатель солнце/луна в шапке, выбор сохраняется между сессиями
УлучшеноГраф в светлой теме — фон, узлы и подписи рёбер корректно перекрашиваются при смене темы
ИсправленоБейдж «Исход.» — исходящие транзакции теперь отмечены красным цветом, входящие — зелёным
ИсправленоФон подписей рёбер — в светлой теме фон подписей на рёбрах графа больше не чёрный
Версия 2.1 — Синтетические данные
НовоеСинтетика — генерация датасета по диапазону дат: выбери цепочку, период и количество транзакций
НовоеРавномерная выборка — транзакции берутся по одной с каждого дня периода, а не только с последнего
НовоеБаланс в синтетике — для каждого адреса автоматически подтягивается реальный баланс кошелька
ИсправленоАдрес контрагента — теперь показываются все адреса через « | », без обрезки «+N»
Версия 2.0 — Основа
НовоеПул кошельков — вставь список адресов и получи датасет всех транзакций в формате CSV
УлучшеноГраф транзакций — кнопки + и − на каждом узле для раскрытия цепочки вперёд и назад
УлучшеноТаблица — сортировка по дате, типу, сумме и контрагенту
ИсправленоСуммы на рёбрах графа отражают реальную долю кошелька в транзакции (UTXO-модель)
